# Scans API

The Scans API lets you create accessibility, performance, and security scans for any public URL. Scans run asynchronously -- you submit a URL, receive a scan ID, and poll for results.

---

## Create a Scan

<span className="method-post">POST</span> `/api/v1/scans`

Creates a new scan for the specified URL. The scan runs asynchronously and typically completes within 15-60 seconds depending on page complexity and the number of checks selected.

### Request Body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `url` | string | Yes | The URL to scan. Must be a valid, publicly accessible URL. |
| `checks` | string[] | No | Array of check types to run. Default: `["accessibility"]`. Options: `"accessibility"`, `"performance"`, `"security"`, `"seo"`. |
| `pages` | number | No | Maximum number of pages to crawl. Default: `1`. Max: `50` (varies by plan). |
| `projectId` | string | No | Associate this scan with a project for tracking. |
| `waitForResult` | boolean | No | If `true`, the request blocks until the scan completes (up to 120s). Default: `false`. |
| `viewport` | object | No | Custom viewport size. Default: `{ "width": 1280, "height": 720 }`. |

### Response

```json
{
  "success": true,
  "data": {
    "scanId": "scan_8f3k2j1m5n7p",
    "status": "pending",
    "url": "https://example.com",
    "checks": ["accessibility", "performance"],
    "pages": 1,
    "projectId": null,
    "createdAt": "2025-01-15T10:30:00.000Z"
  }
}
```

### curl

```bash
curl -X POST https://api.preship.dev/api/v1/scans \
  -H "Content-Type: application/json" \
  -H "X-API-Key: sk_live_your_api_key_here" \
  -d '{
    "url": "https://example.com",
    "checks": ["accessibility", "performance", "seo"],
    "pages": 5,
    "projectId": "proj_abc123"
  }'
```

### JavaScript

```javascript
const response = await fetch("https://api.preship.dev/api/v1/scans", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "X-API-Key": process.env.PRESHIP_API_KEY,
  },
  body: JSON.stringify({
    url: "https://example.com",
    checks: ["accessibility", "performance", "seo"],
    pages: 5,
    projectId: "proj_abc123",
  }),
});

const { data } = await response.json();
console.log(`Scan created: ${data.scanId}`);
```

### Python

```python
import requests

response = requests.post(
    "https://api.preship.dev/api/v1/scans",
    headers={
        "Content-Type": "application/json",
        "X-API-Key": "sk_live_your_api_key_here",
    },

        "pages": 5,
        "projectId": "proj_abc123",
    },
)

data = response.json()["data"]
print(f"Scan created: {data['scanId']}")
```

## Get Scan Results

<span className="method-get">GET</span> `/api/v1/scans/:id`

Retrieves the full results of a scan. While the scan is in progress, `status` will be `"pending"` or `"running"`. Once complete, the response includes scores, issues, and detailed findings.

### Path Parameters

| `id` | string | The scan ID (e.g., `scan_8f3k2j1m5n7p`) |

```json
{
  "success": true,

  }

```

### Scan Statuses

| Status | Description |
|--------|-------------|
| `pending` | Scan has been queued and is waiting to start |
| `running` | Scan is actively in progress |
| `completed` | Scan finished successfully |
| `failed` | Scan failed due to an error (e.g., URL unreachable) |

### curl

```bash
curl https://api.preship.dev/api/v1/scans/scan_8f3k2j1m5n7p \
  -H "X-API-Key: sk_live_your_api_key_here"
```

### JavaScript

```javascript
const response = await fetch(
  "https://api.preship.dev/api/v1/scans/scan_8f3k2j1m5n7p",
  {
    headers: {
      "X-API-Key": process.env.PRESHIP_API_KEY,
    },
  }
);

const { data } = await response.json();

if (data.status === "completed") {
  console.log(`Accessibility score: ${data.scores.accessibility}`);
  console.log(`Total issues: ${data.summary.totalIssues}`);

  data.issues.forEach((issue) => {
    console.log(`[${issue.impact}] ${issue.description}`);
  });
}
```

### Python

```python
import requests

response = requests.get(
    "https://api.preship.dev/api/v1/scans/scan_8f3k2j1m5n7p",
    headers={"X-API-Key": "sk_live_your_api_key_here"},
)

data = response.json()["data"]

if data["status"] == "completed":
    print(f"Accessibility score: {data['scores']['accessibility']}")
    print(f"Total issues: {data['summary']['totalIssues']}")

    for issue in data["issues"]:
        print(f"[{issue['impact']}] {issue['description']}")
```

### Polling for Results

If you did not set `waitForResult: true` when creating the scan, poll the status endpoint until the scan completes:

```javascript
async function waitForScan(scanId, apiKey, maxWait = 120000) {
  const startTime = Date.now();
  const pollInterval = 2000;

  while (Date.now() - startTime < maxWait) {
    const response = await fetch(
      `https://api.preship.dev/api/v1/scans/${scanId}`,
      { headers: { "X-API-Key": apiKey } }
    );

    const { data } = await response.json();

    if (data.status === "completed") return data;
    if (data.status === "failed") throw new Error("Scan failed");

    await new Promise((resolve) => setTimeout(resolve, pollInterval));
  }

  throw new Error("Scan timed out");
}

const result = await waitForScan("scan_8f3k2j1m5n7p", process.env.PRESHIP_API_KEY);
```

## List Scans

<span className="method-get">GET</span> `/api/v1/scans`

Returns a paginated list of your scans, sorted by creation date (newest first by default).

### Query Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `projectId` | string | -- | Filter scans by project |
| `status` | string | -- | Filter by status: `pending`, `running`, `completed`, `failed` |
| `page` | number | `1` | Page number (1-indexed) |
| `limit` | number | `20` | Items per page (max 100) |
| `sort` | string | `createdAt` | Sort field: `createdAt`, `completedAt`, `score` |
| `order` | string | `desc` | Sort order: `asc` or `desc` |

### Response

```json
{
  "success": true,
  "data": {
    "items": [
      {
        "scanId": "scan_8f3k2j1m5n7p",
        "url": "https://example.com",
        "status": "completed",
        "checks": ["accessibility"],
        "scores": { "accessibility": 87 },
        "summary": { "totalIssues": 5, "critical": 1, "serious": 2, "moderate": 1, "minor": 1 },
        "projectId": "proj_abc123",
        "createdAt": "2025-01-15T10:30:00.000Z",
        "completedAt": "2025-01-15T10:30:42.000Z"
      }
    ],
    "pagination": {
      "page": 1,
      "limit": 20,
      "totalItems": 47,
      "totalPages": 3,
      "hasMore": true
    }
  }
}
```

### curl

```bash
curl "https://api.preship.dev/api/v1/scans?projectId=proj_abc123&limit=10&page=1" \
  -H "X-API-Key: sk_live_your_api_key_here"
```

### JavaScript

```javascript
const params = new URLSearchParams({
  projectId: "proj_abc123",
  limit: "10",
  page: "1",
  sort: "createdAt",
  order: "desc",
});

const response = await fetch(
  `https://api.preship.dev/api/v1/scans?${params}`,

    headers: { "X-API-Key": process.env.PRESHIP_API_KEY },

);

const { data } = await response.json();
console.log(`Found ${data.pagination.totalItems} scans`);
data.items.forEach((scan) => {
  console.log(`${scan.scanId}: ${scan.url} - Score: ${scan.scores.accessibility}`);
});
```
