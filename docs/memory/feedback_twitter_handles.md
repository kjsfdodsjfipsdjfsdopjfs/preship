When posting on Twitter/X, ALWAYS tag the Twitter handle of every company, person, or product mentioned.

**Why:** Rodrigo was frustrated multiple times that tweets mentioned companies by name without @ handles. Handles are critical for engagement — tagged accounts get notifications, may respond/repost, and it looks more professional.

**How to apply:**
- Before writing any tweet, look up the Twitter handle of every entity mentioned
- Include @ handles inline in the tweet text, not just names
- For the scan report, always tag winners AND losers
- Common handles: @AnthropicAI, @perplexity_ai, @figma, @vercel, @supabase, @PostHog, @Replit, @zapier, @retool, @mercury, @calcom, @deel, @segment, @NotionHQ, @getpostman, @ProductHunt, @ycombinator
